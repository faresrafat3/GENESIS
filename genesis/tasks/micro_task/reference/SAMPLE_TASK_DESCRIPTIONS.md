# Sample Task Descriptions — Micro-Task Economy Research

## Task Type
Open-ended research and analysis (qualitative output)

## What the agent must do
1. Research micro-task platforms available to Egyptian workers
2. Verify credibility with real payment proofs
3. Predict future opportunities where AI hasn't penetrated yet
4. Produce structured Arabic-language report

## Success criteria
- Real, currently-active platforms identified
- Honest about what couldn't be verified (no hallucinated links)
- Egypt availability explicitly stated (not assumed)
- Future predictions grounded in real AI limitations
- Report saved to results/micro_task_report.md

## Example good output excerpt
```
| Clickworker | تصنيف نصوص، تقييم بيانات AI | فورية | $1-3 | ✅ نعم | reddit.com/r/beermoney/... | ✅ مؤكدة |
| Neevo | تدريب نماذج NLP، تسمية بيانات | بعد قبول | $2-5 | ⚠️ غير مؤكدة لمصر | غير موجود | ⚠️ تحقق يدوي |
```

## Notes
- This task has NO automated scoring
- Human evaluator reads the report and judges quality
- Agent should use 3 turns: research → self-critique → refinement
